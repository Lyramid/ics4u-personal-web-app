ICS4U Projects

Below are the two ICS4U assignments I coded

Culminating Project:

Below is the javascript game that I coded for my culminating. It is flash game inspired but the current gameplay is kind of like a visual novel except no choices currently. I definetly overestimated the time it would take to research and pick up javascript, and ended up rushing to get a "working" base that I am happy with.

I coded the app with the kaboom.js library, which acted as a game engine. Learning to utilize it, as well as learning javascript was a big challenge. Trying to figure out how to do things took a lot of time and brain power, causing me to lose motivation in my work on multiple occasions. But powering through with lots of help from google and the kaboom doccumentation allowed me to get the project to a state where I am quite proud of my work. It really captured the essense of what I set out to do, creating something inspired by the flash games of my childhood.

My future plans for the project is to add sound and music to the game, create more levels and add to the gameplay, either through user choices, or other actionable items (stocking or selling inventory for example).

GUI app:

Below is the project I coded in java with my classmate Max for the GUI assignment. Where we had to code an app that utilized a GUI in some way. Our idea is akin to "The Impossible Quiz" which was popular on the internet a while back. Asking similarly unstraight forward questions with what we hoped are not obvious answers. I'm really happy with the varity of question types we have. The biggest challenge we faced when programming it was figuring out how to diferetiate between the user clicking the X button or the cancel button, since the library registers them very similarly. We ended up comprimising and accepting both as correct inputs for the question.

We used the JOptionPane library to create the GUI app. In the future we want to make a custom UI for our app, instead of using the default one, and we also want to add very annoying sound effects when you answer a question wrong (or even when you answer right).
